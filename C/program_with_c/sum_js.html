加總 -- sum.js

範例 -- 加總

使用 while 迴圈

檔案：wsum.js

sum=0;
i=1;
while (i<=10) {
  sum = sum + i;
  console.log("i=", i, " sum=", sum);
  i = i + 1;
}

執行結果：

使用 for 迴圈

檔案：sum.js

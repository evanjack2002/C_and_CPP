string.c

範例：字串 -- string.c

C 語言宣告的字串，其實只是一種以 ASCII 編碼 0 的字母作為結果的字元陣列而已。(不像 javascript 的字串是一個物件，C 語言沒有物件的概念)。

檔案：string.c

#include <stdio.h>
#include <string.h>

int main() {
  int i;
  char s[]="hello!"; 
  for (i=0;i<strlen(s)+1;i++) {
    printf("s[%d]=%c ascii=%d\n", i, s[i], s[i]);
   }
}
